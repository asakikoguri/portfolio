ポートフォリオサイト

自身の転職活動のためのポートフォリオサイトを制作しました

Art Direction / Design / Development

制作時期
2026.4
制作期間
direction 3日
design 3日
cording 1日

ターゲット

採用担当者の方々

目標

忙しい採用担当者の利便性を追求したシンプルな構成と、
書類では伝わりにくい人柄やスキル状況が伝わるデザインを目指しました。

デザイン戦略と設計

トップページは1分で概要が把握できるよう情報を最小限に絞り、詳細は下層ページへ集約。MVにInstagramギャラリー、自己紹介へと繋げることで自身のイメージを持ってもらいその後の内容にも興味を持ってもらう流れにしました。また、JavaScriptを用いた適度な動きを加えることで、シンプルなデザインでも飽きさせない視覚体験と同時に実装スキルを表現しました。

コンセプトとアプローチ

信頼感と親しみやすさを両立させるため、全体を温かみのあるニュアンスカラーで統一。フォントを使い分けることで、情報の重要度を直感的に整理しました。モックアップも一貫性のあるシンプルなデザインで統一することで、世界観の統一を図っています。

使用ツール

Figma　/　HTML　/　CSS　/　Javascript
